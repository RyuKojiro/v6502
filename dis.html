Disassembler

Arguments and Usage

usage: dis6502 [-o out_file] [-F format] [file ...]

Any number of binaries may be specified and they will all be disassembled, individually.

  • -o out_file

    Specify an out_file to write the assembly output to. If none is specified, the output will be piped to stdout.

  • -F format

    Specify a format to use when reading in binaries.

    Supported formats:

    • flat - Root segment only, no symbol table, single execution unit ready to be loaded into memory for running.
    • ines - The iNES ROM format, which is the de facto standard for compiled NES ROMS.

Notes

Depending on how well engineered/accurate the ROM/header information is in an NES ROM, the program code may overrun into the CHR ROM, or might have padding which will be assembled inline (but should not hinder disassembler byte alignment.)
